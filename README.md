# CERCAPP APP CLIENT

Esta es la app cliente de cercapp, el frontend. De momento se sube como archivo estático dentro de la API, aprovechando la funcionalidad de fastapi para ello.

## Stack
- Vite
- Vue
- Typescript
- Tailwind

## Puntos o fucniones a desarrollar en el futuro:
- Dark/Light mode
- Notificaciones push sincronizadas desde el backend o que se disparen con cambios o acciones desde el backend
- Integración con funciones de KYC pendientes
- Crear vistas y modales para gestión de pagos con pasarela de pagos (depende del desarrollo de esos puntos en el backend)
